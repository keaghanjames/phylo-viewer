PHYLOSCOPE — PATCH NOTES
========================

[v0.2] — 2026-04-23
--------------------
- Added "What's new in v0.2" welcome modal (shown once per version, localStorage-gated)
- Added "How to Cite" button with APA and BibTeX citation entries (Copy buttons included)
- Added Lineages Through Time (LTT) plot (LTT Plot button, visible when tree has branch lengths)
  - Step-function plot: X = time from root, Y = lineage count (log scale)
  - Opens in a modal; handles polytomies and non-ultrametric trees
- Renamed "Radial" layout button to "Radial (Beta)"
- Arrow keys (↑ ↓ ← →) now pan the canvas view
- Tip rename: selecting a leaf in the Clade tab shows a rename input field
- Added trait data visualisation (third sidebar tab: Traits)
  - Trait squares appear between the tip node and tip label; labels dynamically
    shift right based on the number of active traits
  - Upload a CSV file (tip names in column 1, traits in remaining columns)
  - Numeric columns auto-detected as continuous (Viridis gradient)
  - Text columns auto-detected as discrete (categorical colour palette)
  - Toggle each trait on/off; colour legend shown per trait
  - Missing data / NA cells rendered as grey squares, never crash
  - Rectangular layout: coloured squares next to each tip label
  - Radial layout: tip node dots coloured by first active trait
  - Shows matched vs total tip count after upload
- Added Weight slider to scale branch line width and node dot size together
- Colour-coded Branch Info panel: branch length in orange, clade stats in blue
- Paraphyletic group panel now shows MRCA node height alongside Phylogenetic Diversity


[v0.1.2] — 2026-04-23
----------------------
- Added "See Source" button (visible when a tree is loaded)
  - Opens a modal showing the full Newick string for the current tree
  - User can edit the text directly and hit Confirm to reload the tree
  - Includes a Copy button for quickly grabbing the Newick string
- Fixed: switching between Rectangular and Radial layouts now resets the
  canvas to the centred position (previously the old pan/zoom offset was
  kept, which could leave the tree off-screen)
- Renamed "Load Sample" button to "Demo Tree"


[v0.1.1] — 2026-04-23
----------------------
- Added feedback form (Feedback button in header)
  - Collects email address, feedback type (bug / feature / general), and message
  - Submits to Formspree (xgorpzja); responses stored in dashboard and emailed on receipt
- Updated header title to "Phylo Viewer v0.1"


[v1.1.0] — 2026-04-23
----------------------
- Replaced Vite boilerplate favicon with cartoon Darwin "I think" tree icon
  (green rounded square, white tree silhouette capturing trunk, basal node
  circle, lower-right A branch, left D branches, and upper B/C canopy)
- Updated app header: replaced 🌿 emoji with the Darwin tree SVG icon
- Updated README with live app URL and description of what the tool does


[v1.0.0] — 2026-04-22
----------------------
- Initial release
- Vite + React + D3 project scaffolded and deployed to GitHub Pages
  at https://keaghanjames.github.io/phyloscope/
- Core phylogenetic tree viewer features:
  - Load trees from .nwk / .tre files or paste Newick string directly
  - Rectangular and radial layouts
  - Branch-length scaling with time axis and flip (time before present)
  - Zoom, pan, snap-to-node navigation; Space to re-centre
  - Clade selection (click), MRCA selection (Shift+click),
    paraphyletic group (Cmd/Ctrl+click)
  - Clade collapse, extract, delete
  - Paraphyletic group Phylogenetic Diversity calculation
  - Ladderize (ascending / descending)
  - Reroot on any node
  - Undo (20-step history)
  - Export to Newick (.nwk)
  - Tip label search with zoom-to-result
  - Toggle tip labels on/off; adjustable font size
  - GitHub Actions CI/CD for automatic deployment on push to main
